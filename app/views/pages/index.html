Hi I'm Daniel Schur!

I was born and raised in Melbourne, Fl.

I moved to Miami in 2010 to attend school at the University of Miami where I studied Graphic Design, Marketing, and Psychology.

After graduating I wanted to challenge myself further and continue my education at Wyncode Academy, where I graduated with the skills to call myself a Junior Fullstack Developer.

While at Wyncode Academy I worked on a few different projects. See my journey from total newbie to Full Stack Developer below!

My First Application: Repl Game

Deus Ex was the first real ruby application I made.

I used some basic logic, a logic tree to check the players sequence of decisions, and a module to save all the possible dialogue answers.

My introduction to classes and testing

Quad is a simple ruby application that was created to familiarize myself with object oriented programming, classes, and testing.

Javascript, the quirky cousin

In this application I used javascript to take on the classic fizzbuzz question

Let's get responsive!

This was a landing page I created to experiment with using HTML, CSS, and Boostrap become familiar with responsive design.

Rails and API integration

Wynning was an application that I developed with some teammates that uses rails and integrates the instagram API to show off the most recent photos associated with curated and custom hashtags.

The Final Project!

Foodnag is a rails application that uses a large array of technologies, from ajax requests to rake tasks. I pitched and presented this project to over 300 members of the Miami Tech community when I graduated from Wyncode and am very proud of it.
